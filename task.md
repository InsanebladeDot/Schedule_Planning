📱 基于 UniApp + JS 的粉白色系日程任务表开发任务清单 (含图标)
1. 项目初始化与全局配置
创建项目：使用 HBuilderX 创建新项目，命名为 "PinkTask"。
配置页面：在 pages.json 中注册三个页面：weekly (周期表)、timeline (时间轴)、profile (我的)。
配置 TabBar：
首页 (周期表)：iconfont 图标 日历 (未选中：浅粉 #FFB6C1，选中：亮粉 #FF69B4)。
时间轴：iconfont 图标 时钟 (未选中：浅粉，选中：亮粉)。
我的：iconfont 图标 用户 (未选中：浅粉，选中：亮粉)。
引入样式：定义全局 SCSS 变量，设置粉白主题色、圆角 (16rpx) 和阴影。
2. 页面 1：周期表 (Weekly View)
顶部栏：标题“本周日程”，右侧操作图标使用 iconfont 添加 (加号)。
星期选择器：
使用横向滚动视图。
每一项使用圆形卡片，内部包含 iconfont 周一 到 周日 (建议使用简单的数字1-7或英文缩写)。
当前选中项应用渐变背景 (从 #FFB6C1 到 #FF69B4)。
任务项：
左侧复选框：未选中时为描边圆圈 (iconfont 圆形 空心)，选中时填充为亮粉色并显示 iconfont 对勾。
右侧内容：任务标题，下方小字显示时间 (iconfont 时间 图标 + 文字)。
滑动操作：左滑显示删除按钮 (iconfont 垃圾桶)，右滑显示完成按钮 (iconfont 完成)。
3. 页面 2：时间轴 (Timeline View)
左侧时间轴：固定宽度，显示 00:00 - 24:00 的刻度。
右侧事件卡片：
卡片左侧添加一条宽约 6rpx 的色块，颜色使用 #FFB6C1。
卡片内顶部显示 iconfont 闹钟 图标，右侧显示具体时间。
卡片标题和描述。
空状态：当没有任务时，显示 iconfont 空状态 (如：一个日历上画个问号) 并提示“暂无日程”。
4. 页面 3：我的/设置 (Profile/Settings)
头部：显示用户头像 (默认使用 iconfont 用户 头像) 和昵称。
统计卡片：使用 iconfont 列表 图标表示任务总数，对勾 图标表示已完成数。
设置列表：
第一项：设置 (iconfont 齿轮) -> 进入详情。
第二项：黑夜模式 (iconfont 月亮) -> 点击切换。
第三项：关于 (iconfont 信息)。
底部导航：保持与首页一致。
5. 核心逻辑与数据持久化
数据模型：
id: 唯一标识。
title: 任务标题。
time: 时间 (HH:mm)。
done: 是否完成 (Boolean)。
repeat: 重复周期 (Array: [1,2,3...] 对应周一到周日)。
date: 具体日期 (用于时间轴视图)。
本地存储：封装 storage.js，使用 uni.setStorageSync 保存任务数组。
页面通信：使用 Vuex 或全局事件总线，确保在 A 页面添加/修改任务后，B 页面能实时更新。
6. 异常处理与交互优化
网络异常：虽然主要是本地应用，但需处理本地存储读写失败的情况 (try-catch)。
滑动冲突：在时间轴页面，确保垂直滑动和水平滑动 (如删除按钮) 不冲突。
输入验证：添加任务时，标题不能为空，时间格式必须正确。
删除确认：点击删除时，弹出 uni.showModal 确认框，防止误触。
7. 黑夜模式切换
CSS 变量：在 App.vue 的 <style> 中定义两套 CSS 变量 (白天/黑夜)。
切换逻辑：在“我的”页面点击 月亮 图标，通过 JavaScript 动态修改 document.documentElement.style.setProperty 来切换主题色。
图标适配：确保 iconfont 图标在暗黑背景 (如 #1A1A1A) 下依然清晰可见，必要时调整图标颜色为白色或浅粉。
8. 画面二次细化 (精致且简洁)
图标统一：全应用使用同一套 iconfont 图标库 (推荐 "阿里巴巴普惠体" 配套图标或 "IconPark" 线性图标)，确保线条粗细一致 (建议 2px)。
微交互：
点击 TabBar 时，图标有轻微的“跳动”动画。
任务完成时，文字有“划线”动画，颜色渐变。
页面切换时，使用淡入淡出或滑动入场动画。
留白与排版：
卡片之间留有足够的间距 (至少 20rpx)。
字体层级分明：标题 (加粗 32rpx)，正文 (常规 28rpx)，辅助信息 (细体 24rpx，颜色 #999)。
移除所有不必要的分割线，利用卡片阴影和背景色差来区分层级。
